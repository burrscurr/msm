## Mitmachen

Das Skript wird mit `pdflatex` erstellt. Am einfachsten wird das Makefile
verwendet:

```
cd src
make
```

Erweiterungen und Korrekturen des Skripts sind sehr erwünscht und können als
Issues/Pull requests erstellt werden.
